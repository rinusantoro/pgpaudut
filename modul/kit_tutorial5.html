Kit Tutorial

Kit tutorial adalah seperangkat rancangan digunakan saat tutorial, yang terduri dari RAT, SAT, kisi-kisi tugas, pedoman tugas dan pedoman penskoran tugas. RAT SAT tutorial tatap muka dengan tuton memiliki beberapa perbedaan.

A.     RAT SAT TUTORIAL TATAP MUKA (TTM) ATAU TUWEB

 Contoh RAT SAT TTM/tuweb untuk mata kuliah PAUD4204 Media dan Sumber Belajar TK adalah sebagai berikut.

RANCANGAN AKTIVITAS TUTORIAL (RAT)

PROGRAM STUDI PG PAUD

Nama Mata Kuliah                             : Media dan Sumber Belajar TK
Kode Mata Kuliah                              : PAUD4204
Jumlah sks                                        : 3 sks
Nama Pengembang                          : Pasiningsih, M. Ed
Nama Penelaah                                : Dr. Mukti Amini, M. Pd

Deskripsi Singkat Mata kuliah          : Mata kuliah Media dan Sumber Belajar TK (PAUD 4204) memiliki bobot 3 SKS. Mata kuliah ini membekali mahasiswa agar memiliki kemampuan dalam memanfaatkan, merancang serta mengembangkan media dan sumber belajar untuk pembelajaran di lembaga PAUD.

Media dan Sumber Belajar TK (PAUD 4204) membahas hakikat media dan sumber belajar dengan anak usia dini, jenis-jenis sumber belajar, perencanaan, pembuatan dan pengelolaan media dan sumber belajar termasuk Alat Permainan Edukatif (APE), serta pemanfaatan lingkungan sebagai sumber belajar pada anak usia dini. Di samping mempelajari Buku Materi Pokok (BMP) sebagai sumber belajar utama, mahasiswa juga belajar melalui sumber belajar pendukung, berupa PPT sebagai materi inisasi, video, artikel, dan gambar. Keberhasilan mahasiswa akan diukur dengan nilai tutorial dimana mahasiswa akan merancang suatu media pembelajaran /APE dan praktek mengajar dengan menggunakan media pembelajaran/ APE yang dikembangkan sebagai bagian tugas tutorial dan nilai di Ujian Akhir Semester.

 

Capaian Pembelajaran Mata Kuliah   :  Setelah mengikuti mata kuliah Media dan Sumber Belajar TK (PAUD 4204), mahasiswa diharapkan terampil dalam memanfaatkan, merancang serta mengembangkan media dan sumber belajar untuk pembelajaran di lembaga PAUD.


Tahun Pengembangan                      : 2020
